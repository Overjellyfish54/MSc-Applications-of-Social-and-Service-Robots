This project explores the use of a humanoid robot (NAO) as an assistant physiotherapist, focusing on one or more of the following areas:

 - Upper-limb rehabilitation
 - Whole-body collaborative rehabilitation
 - Lower-limb massage and therapy

The project is developed using a Design Research approach (Understand, Empathise, Define, Ideate, Prototype). Research is conducted to identify current healthcare and physiotherapy challenges, which then inform the design of an innovative humanoid robotics program grounded in up-to-date global literature.
The outcome is a 3–5 minute human–robot interactive program addressing a real-world rehabilitation problem. The system incorporates sensor inputs (such as tactile, pressure, sonar, microphone, or infrared sensors), robotic vision (dual cameras for recognition, localisation, or motion planning), and physical outputs including movement, speech, LED animations, music, and expressive robot behaviours. Creative elements such as props, music, or dance may also be included.

All programs and program versions of the robot physiotherapy assitant prototype made for the ENG7007 Applications of Social and Service Robots module by Dan Birjaru, Ross Smith, Callum Waters and Elwin Lewin. Programs with "dancopy" are alternate versions made by Dan alternate functionality.
